<h2 id='suzen21'> Suzen21 </h2>

В этом задании нужно удалить только определенные файлы. Для этого будем использовать регулярные выражения.

1. Удаляем все файлы которые содержат *цифры* в своем названии для этого используем шаблон **./\*[0-9]\*.txt**
2. Удаляем все файлы которые содержат *буквы нижнего регистра* в своем названии для этого используем шаблон **./\*[a-z]\*.txt**
3. Удаляем все файлы которые содержат *"test-"* в своем названии для этого используем шаблон **./\*test-\*.txt**

Ну а для удаления самих файлов соответственно используем команду **rm**.
Далее стоит проверить содержание текущей директории командой **ls**. Там мы и найдем наш флаг.

<h2 id='suzen22'> Suzen22 </h2>

Здесь нужно создать 1000 файлов. Для этого мы будем использовать цикл **for** в **bash**. Для получении последовательности, по которой будет происходить перебор, используем команду **seq**. Затем мы перенаправляем вывод при помощи **>** в нужный нам файл. Если его нет  *bash* автоматически создаст его, что нам и нужно.
После этого в текущей директории появится файл, названием которого является флаг. Найдем его при помощи ls. Чтобы не копаться в файлах сразу отфильтруем вывод. Для этого используем команду **egrep**, ведь она позволяет отфильтровать по паттерну. Проделав все это мы получим наш флаг.

![/images/22.png](https://github.com/sergo2048/OS-lab/blob/master/lab1/images/22.png)
